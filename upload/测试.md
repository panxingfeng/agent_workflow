## 🌟 项目概览

AgentChatBot 是一个基于 langchain/Ollama 的智能体框架，支持：

🤖 私人微信部署 (通过 VChat)

💼 飞书机器人集成

🎨 React UI 界面

🛠 自定义工具扩展

## 🚀 核心功能

### 代码生成

基于本地 Ollama 部署

支持多种编程语言

智能代码补全

### 多平台支持

✅ 微信接入

✅ 飞书部署

✅ React UI 界面

🔧 更多平台持续集成中...

## 📢 最新更新

### 2024-10-16
